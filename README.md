🚴 Bike Sales Analysis (Excel Project)
📌 Project Overview

This project analyzes bike sales data using Microsoft Excel to uncover insights related to customer demographics, income, commute distance, and purchasing behavior. The goal is to identify patterns that influence bike purchases and support data-driven business decisions.

📂 Dataset Information

The dataset contains customer information and their bike purchasing status.

Key Columns:
ID – Unique customer ID
Marital Status – Married / Single
Gender – Male / Female
Income – Annual income of customer
Children – Number of children
Education – Education level
Occupation – Job type
Home Owner – Yes / No
Cars – Number of cars owned
Commute Distance – Distance to work
Region – Customer region
Age – Customer age
Purchased Bike – Yes / No (Target variable)
🎯 Objectives
Clean and prepare raw dataset
Perform exploratory data analysis (EDA)
Identify factors affecting bike purchases
Create interactive dashboards in Excel
Visualize trends using charts and pivot tables
🧹 Data Cleaning Steps
Removed duplicate records
Handled missing values
Standardized categorical values (Yes/No, Male/Female)
Converted income and age into proper formats
Created age brackets for better analysis
📊 Analysis Performed
Bike purchase vs income comparison
Gender-wise purchase distribution
Impact of commute distance on buying decision
Age group analysis
Region-wise sales performance
Occupation vs purchase behavior
📈 Key Insights
Higher income groups are more likely to purchase bikes
Middle-aged customers show higher purchase rates
Commute distance significantly affects buying decision
Certain occupations show stronger purchasing trends
Regional differences exist in bike demand
